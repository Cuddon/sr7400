SR7400/SR8400 Surround Sound Receiver Web Service

SR7400 (or SR8400) surround sound amplifier web service using a Global Cache GC-100 to connect to the SR7400/SR8400 serial port via TCP.

API structure

/api/request_type/request_string              --> returns string
/api/request_type/request_string?return=json  --> returns json (optional) ** Not yet implemented **

Device commands and status requests

/api/command/turn_power_on
/api/command/get_power_status

Macros

/api/macros/watch_tv_with_surround_sound

/api/macro/run/commands (Not yet implemented)- commands is a list of commands

Configuration

/api/config/settings
/api/config/macros
/api/config/protocol
/api/config/mappings
/api/config/help

Logs

/api/log/requests
/api/log/errors
